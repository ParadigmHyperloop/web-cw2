Help

                      
                  

                  

                    

                      Welcome to the Paradigm Pod Control UI.  This Web UI is designed to provide an easy to use interface for
                      performing normal operational interactions with the pod.  This includes:
                    


                    
    
                      
  •  Filling the HP pressure system 
    • 
                      
  •  Arming the pod 
    • 
                      
  •  Controlling the pod's low power mode 
    • 
                      
  •  Engaging and Disengaging both of the battery packs. 
    • 
                      
  •  Asserting an Emergency remotely 
    • 
                      
  •  Remotely shutting down the pod 
    • 
                      
  •  Viewing the state of the air supply 
    • 
                      
  •  Viewing the state of the batteries 
    • 
                      
  •  Viewing the state of the network connection to the pod 
    • 
                      
  •  Viewing the state of the thermal monitoring sensors 
    • 
                      
  •  Viewing master caution and master warning alarms 
    • 
                    

                    

                      For a full demonstration of the capabilities of this application please contact Dan Hartman, Janna Shaftan,
                      and Ed Hurtig of the Paradigm Controls Team
